# Food Price Analysis in India (1994–2019)

## Project Overview
This project analyzes historical food price trends in India using data from the
World Food Programme (WFP).

## Analysis Performed
- Data cleaning and preprocessing
- Trend analysis of major commodities
- Time-series visualization
- Price volatility study

## Dataset
- Source: World Food Programme (WFP)
- Country: India
- Time Period: 1994–2019
- Commodities: Rice, Wheat, Sugar

## Key Highlights
- Performed Exploratory Data Analysis (EDA) on Indian food price data (1994–2019).
- Cleaned and transformed large CSV data using Pandas, handling date parsing and metadata rows.
- Analyzed long-term price trends of Rice, Wheat, and Sugar.
- Visualized price trends and volatility using Matplotlib.
- Identified inflationary patterns in Indian food markets.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook
